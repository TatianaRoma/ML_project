The description of files in this folder:

Yelp_Review_Rating_Prediction.ipynb --> contains the code and output for training and 
testing the final pipelines and comparing their performance.

Yelp_1000000_reviews_MLR.ipynb --> contains the code and output for parameter grid search 
for Logistic Regression

Yelp_1000000_reviews_MNB_SVM.ipynb --> contains the code and output for parameter grid 
search for Multinomial Naive Bayes and Support Vector Machines.

RunningTimesExperiment.ipynb --> contains the code and output  to test how long each 
algorithm takes and how the accuracy improves with more data.
